Uninstalling previous versions¶

It is no longer necessary to uninstall previous versions of OPF before installing a new version. The installation utility will backup existing data files and install new files. However, to be safe please save your own copy of your data files. If, for any reason, you still want to uninstall the program please follow the instructions below:

Under Windows¶

In Windows open the ‘Control Panel’ from the main menu, double click ‘Programs and Features’, (previously known as ‘Add Programs’) and click on the ‘OPF’ entry. Select ‘Uninstall’ or ‘Remove’. When OPF has been removed close ‘Programs and Features’ and close the ‘Control Panel’

Under Linux¶

Delete the files in the installation directory. If you have any desktop shortcuts or menu links delete those as well.
